Install virtual env
  >>sudo pip install virtualenv
  
  >>mkdir ~/.venvs
  (create a folder in home directory to store all your virtual environments)

  >>virtualenv --system-site-packages ~/.venvs/lpthw
  (run virtual environment and include system site packages, instruct to build virtual env in .venvs/lpthw
  
  >>./ ~/.venvs/lpthw/bin/activate
  (source, and enter the virtual env)
